## How to Practice Solo

1. Open a problem folder (e.g., `Excel Problem 1/`)
2. Read `problem.md` for context and questions
3. Import `data.csv` into Excel
4. Answer the questions using:
   - PivotTables
   - Charts and visualizations
   - Formulas (SUM, AVERAGE, IF, VLOOKUP, etc.)
5. Document your insights

## Peer Interview Format

### Setup (5 min)
- **Interviewer**: Select a problem
- **Candidate**: Open Excel, import the dataset
- Share screen for collaboration

### Interview (30 min)
1. Read the case study context together
2. Ask questions one at a time
3. Encourage candidate to:
   - Explore data first (structure, missing values)
   - Create visualizations
   - Explain their reasoning

### Skills to Evaluate
- Data cleaning and preparation
- PivotTable creation and usage
- Chart selection and design
- Formula proficiency
- Business insight generation

## Tips for Success

- **Explore first**: Understand the data before answering questions
- **Visualize**: Charts often reveal insights faster than tables
- **Be methodical**: Clean data → Analyze → Summarize
- **Think business**: What actionable insights can you provide?
