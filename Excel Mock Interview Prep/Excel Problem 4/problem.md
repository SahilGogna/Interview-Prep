# Excel Problem 4: Call Center Performance Analysis

**Difficulty:** Easy

## Context

You are working as a data analyst for TechSupport Inc., a customer service company. The operations manager wants to understand call center performance and identify areas for improvement.

You have been given 6 months of call data to analyze.

**Dataset**: [data.csv](https://github.com/SahilGogna/Interview-Prep/blob/main/Excel%20Mock%20Interview%20Prep/Excel%20Problem%204/data.csv)

---

## Dataset Information

| Column | Description |
|--------|-------------|
| Call_ID | Unique identifier for each call |
| Date | Date of the call (YYYY-MM-DD) |
| Time | Time the call was received (HH:MM) |
| Agent | Name of the agent who handled the call |
| Call_Type | Category of the call (Billing, Technical Support, etc.) |
| Wait_Time_Seconds | How long the customer waited before being connected |
| Call_Duration_Seconds | Total length of the call |
| Resolution | Outcome of the call (Resolved, Escalated, etc.) |
| Satisfaction_Score | Customer rating from 1 (poor) to 5 (excellent) |

---

## Tasks

Import the dataset into Excel and answer these questions:

1. What is the **average wait time** and **average call duration** across all calls?

2. Which **agent** has the highest resolution rate? Which has the highest average satisfaction score?

3. Are there **peak hours** when call volume is highest? How does wait time vary by hour?

4. Which **call types** take the longest to resolve? Which have the lowest satisfaction scores?

5. Is there a **correlation** between wait time and customer satisfaction?

6. What **recommendations** would you give to the operations manager to improve customer satisfaction?
