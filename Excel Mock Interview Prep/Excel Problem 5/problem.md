# Excel Problem 5: Investment Portfolio Analysis

**Difficulty:** Medium

## Context

You are a financial analyst at WealthWise Advisors. A client has asked you to analyze their investment portfolio to understand their holdings, performance, and risk exposure.

You have been given their transaction history spanning approximately 18 months.

**Dataset**: [data.csv](https://github.com/SahilGogna/Interview-Prep/blob/main/Excel%20Mock%20Interview%20Prep/Excel%20Problem%205/data.csv)

---

## Dataset Information

| Column | Description |
|--------|-------------|
| Transaction_ID | Unique identifier for each transaction |
| Date | Date of the transaction (YYYY-MM-DD) |
| Ticker | Stock symbol (e.g., AAPL, MSFT) |
| Company_Name | Full company name |
| Sector | Industry sector (Technology, Healthcare, etc.) |
| Action | Buy or Sell |
| Quantity | Number of shares traded |
| Price_Per_Share | Price at which shares were traded |
| Total_Value | Total transaction value (Quantity Ã— Price) |
| Dividend_Yield_Pct | Annual dividend yield percentage |

---

## Tasks

Import the dataset into Excel and answer these questions:

1. Calculate the **total amount invested** (sum of all Buy transactions) and **total amount withdrawn** (sum of all Sell transactions).

2. What is the **current portfolio allocation by sector**? Create a visualization showing the distribution.

3. Which **stocks** does the client hold the most shares in? Which have the highest total investment value?

4. Analyze the **investment pattern over time**. Are there months with higher buying or selling activity?

5. Calculate the **weighted average dividend yield** of the portfolio based on current holdings.

6. What is the **sector concentration risk**? Is the portfolio well-diversified or heavily weighted in certain sectors?

7. Based on your analysis, what **recommendations** would you give to improve portfolio diversification and reduce risk?
